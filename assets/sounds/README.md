# Sound Assets

This folder should contain the following sound files:

## Sound Effects
- `block_place.wav` - Sound when block is placed normally
- `perfect.wav` - Sound when block is placed perfectly
- `game_over.wav` - Sound when game ends
- `button_click.wav` - UI button click sound
- `block_drop.wav` - Sound when block falls off

## Background Music
- `background_music.ogg` - Looping background music

## Getting Free Sound Effects

You can get free sound effects from:
- [Freesound.org](https://freesound.org)
- [OpenGameArt.org](https://opengameart.org)
- [Kenney.nl](https://kenney.nl/assets)

Make sure to check the license for each sound file.

## Supported Formats
- WAV (recommended for short sound effects)
- OGG (recommended for music)
- MP3 (also supported)
